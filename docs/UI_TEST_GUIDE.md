## 수동 테스트 체크리스트

### 1. 홈 화면 테스트

**접속**: http://localhost:8080

#### 확인 사항
- [ ] **배경 그라디언트**: 어두운 배경 (검은색 → 진한 남색)
- [ ] **타이틀**:
  - [ ] "더 마인드" - 보라색 그라디언트 텍스트
  - [ ] "The Mind" - 회색 서브타이틀
  - [ ] 애니메이션 (fadeIn + scale)
- [ ] **새 게임 섹션**:
  - [ ] 반투명 카드 컨테이너
  - [ ] 아이콘 (add_circle_outline)
  - [ ] 버튼 3개 (2명, 3명, 4명)
  - [ ] 보라색 버튼 배경
  - [ ] 아이콘 (people_outline, people, groups)
- [ ] **구분선**: "또는" 텍스트
- [ ] **방 참가 섹션**:
  - [ ] 반투명 카드 컨테이너
  - [ ] 아이콘 (login)
  - [ ] 입력 필드 (방 코드)
  - [ ] 참가하기 버튼
- [ ] **애니메이션**: 순차적 등장 (stagger)

#### 테스트 액션
1. **방 생성**: "2명" 버튼 클릭
   - 로비 화면으로 이동 확인
   - 6자리 방 코드 생성 확인

---

### 2. 로비 화면 테스트

#### 확인 사항
- [ ] **배경 그라디언트**: 홈과 동일
- [ ] **헤더**:
  - [ ] 뒤로가기 버튼 (왼쪽)
  - [ ] "대기실" 제목 (중앙)
- [ ] **방 코드 카드**:
  - [ ] 보라색 그라디언트 배경
  - [ ] 글로우 효과
  - [ ] 열쇠 아이콘
  - [ ] 6자리 코드 (큰 글씨, 6자 간격)
- [ ] **플레이어 섹션**:
  - [ ] people 아이콘
  - [ ] "플레이어" 제목
  - [ ] 인원 표시 (예: 1/2) - 황금색 배지
- [ ] **플레이어 카드**:
  - [ ] 첫 번째 플레이어: 👑 왕관 아이콘
  - [ ] 현재 플레이어: 보라색 테두리
  - [ ] 준비 상태: 초록색 배지 또는 회색 아이콘
- [ ] **준비 버튼**:
  - [ ] 보라색 배경 (준비 전)
  - [ ] 회색 배경 (준비 후)
  - [ ] 체크/닫기 아이콘
- [ ] **게임 시작 버튼** (방장만):
  - [ ] 초록색 배경
  - [ ] play_arrow 아이콘
  - [ ] 모두 준비 시 활성화

#### 테스트 액션
1. **이름 입력**: 다이얼로그에 이름 입력
2. **준비 버튼**: 클릭하여 상태 변경
3. **새 탭**: 같은 방 코드로 2번째 플레이어 참가
4. **게임 시작**: 모두 준비 후 시작 버튼 클릭

---

### 3. 게임 화면 테스트

#### 확인 사항
- [ ] **배경 그라디언트**: 일관된 어두운 테마
- [ ] **상단 컨테이너**:
  - [ ] 반투명 배경
  - [ ] 뒤로가기 버튼 (왼쪽)
  - [ ] **레벨 배지**:
    - [ ] 보라색 그라디언트
    - [ ] 둥근 모서리
    - [ ] 글로우 효과
    - [ ] "레벨 N" 텍스트
  - [ ] 생명 표시 (하트 아이콘)
  - [ ] 수리검 표시 (별 아이콘)
- [ ] **플레이어 상태**:
  - [ ] 카드 형태
  - [ ] 이름 + 남은 카드 수
- [ ] **카드 덱 영역**:
  - [ ] "플레이된 카드" 라벨
  - [ ] **카드 디자인**:
    - [ ] 크기: 70x100
    - [ ] 화이트 그라디언트 배경
    - [ ] 황금색 테두리 (3px)
    - [ ] 글로우 효과
    - [ ] 중앙 숫자 (32px, 굵게, 그림자)
    - [ ] 모서리 숫자 (왼쪽 위, 오른쪽 아래)
  - [ ] 빈 덱: 점선 테두리
- [ ] **수리검 버튼** (있을 경우):
  - [ ] 황금색 배경
  - [ ] 별 아이콘
  - [ ] "수리검 사용 제안" 텍스트
- [ ] **내 손패 영역**:
  - [ ] 반투명 배경
  - [ ] 둥근 모서리 (위쪽만)
  - [ ] 보라색 상단 테두리
  - [ ] person 아이콘 + "내 카드"
  - [ ] 가로 스크롤 카드 리스트
  - [ ] 카드 간격: 16px

#### 테스트 액션
1. **카드 내기**: 손패에서 카드 클릭
   - 카드가 중앙 덱으로 이동
   - 애니메이션 확인 (fadeIn + scale)
2. **수리검 사용**: 버튼 클릭
   - 다이얼로그 표시
   - 투표 UI 확인
3. **레벨 완료**: 모든 카드 정상 순서로 플레이
   - 축하 오버레이 확인
   - 다음 레벨로 진행
