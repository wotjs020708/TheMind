## 🎮 상호작용 테스트

### 필수 테스트 시나리오

#### 1. 방 생성 및 참가
1. Home에서 "2명" 버튼 클릭
2. 이름 입력 다이얼로그에 "테스터1" 입력
3. Lobby 화면 확인
4. **새 브라우저 탭** 열기
5. 동일한 방 코드 입력하여 참가
6. 두 번째 플레이어 이름 "테스터2" 입력
7. 실시간 동기화 확인 (양쪽 탭에서 플레이어 표시)

#### 2. 게임 시작
1. 양쪽 플레이어 "준비" 버튼 클릭
2. 호스트 탭에서 "게임 시작" 버튼 활성화 확인
3. "게임 시작" 클릭
4. 게임 화면으로 전환 확인
5. 카드 분배 확인 (레벨 1 = 각 1장)

#### 3. 카드 플레이
1. **손패에서 카드 클릭**
2. **애니메이션 확인**: FadeIn + Scale
3. 카드가 중앙 덱으로 이동
4. **카드 스타일 확인**:
   - 70x100 크기
   - 황금색 테두리 + 글로우
   - 중앙 숫자 굵게
   - 모서리 숫자 표시
5. 플레이어 카드 카운트 감소

#### 4. 실시간 동기화
1. 한 탭에서 카드 플레이
2. 다른 탭에서 즉시 반영 확인
3. 생명/수리검 카운트 동기화
4. 레벨 진행 동기화

#### 5. 레벨 완료
1. 모든 카드 순서대로 플레이
2. 축하 오버레이 표시
3. 다음 레벨로 진행 (레벨 2 = 각 2장)

#### 6. 실패 시나리오
1. 잘못된 순서로 카드 플레이 (큰 숫자 먼저)
2. 생명 1개 감소 확인
3. 하트 아이콘 개수 변화 확인
4. 양쪽 탭에서 동기화 확인

3. **Unused fields** (2개):
